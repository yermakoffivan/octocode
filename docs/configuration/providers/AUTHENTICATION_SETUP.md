# Authentication Setup

> How to authenticate Octocode MCP with GitHub.

## Provider Setup

GitHub is the only supported provider. Set up authentication step by step (tokens, GitHub Enterprise, clone-tool setup) in the [GitHub Setup Guide](https://github.com/bgauryy/octocode-mcp/blob/main/docs/configuration/providers/GITHUB_SETUP_GUIDE.md).

For all configuration options (env vars, `.octocoderc`, tool filtering, network tuning), see the [Configuration Reference](https://github.com/bgauryy/octocode-mcp/blob/main/docs/configuration/CONFIGURATION_REFERENCE.md).

For available tools and usage, see the [GitHub Tools Reference](https://github.com/bgauryy/octocode-mcp/blob/main/docs/dev/reference/GITHUB_TOOLS_REFERENCE.md).
