## Answer

- `redirect()` is defined in `packages/next/src/client/components/redirect.ts` at line **38** (`export function redirect(`).
- It interrupts rendering by **throwing a `RedirectError`**. The throw line is:
  ```
  throw getRedirectError(url, type, RedirectStatusCode.TemporaryRedirect)
  ```
  (line 45). `getRedirectError()` constructs an `Error` whose `.digest` is set to `` `${REDIRECT_ERROR_CODE};${type};${url};${statusCode};` ``.
- The thrown signal is caught in `packages/next/src/server/app-render/app-render.tsx`. At line **3985** the catch block checks `} else if (isRedirectError(err)) {` and builds the HTTP redirect response:
  ```
  res.statusCode = getRedirectStatusCodeFromError(err)
  ...
  const redirectUrl = addPathPrefix(getURLFromRedirectError(err), basePath)
  ...
  setHeader('location', redirectUrl)
  ```
  This is inside the error-recovery classification block (comment `// MARK: errorRecovery classification`) within the main `renderToHTMLOrFlight` render function in `app-render.tsx`.
