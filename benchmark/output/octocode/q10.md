## Answer

- **Workflow file**: `.github/workflows/turbopack-benchmark.yml`
- **Workflow name**: `Turbopack Benchmark`

- **Trigger events and path filters** (verbatim YAML):
  ```yaml
  on:
    workflow_dispatch:
    push:
      branches:
        - canary
    pull_request:
      types: ['opened', 'reopened', 'synchronize', 'labeled']
      paths:
        - '**/crates/**'
        - '**/Cargo.toml'
        - '**/Cargo.lock'
  ```

- **Benchmark jobs**:

  | Job ID | Name | Cargo package / bench target |
  |--------|------|------------------------------|
  | `benchmark-small-apps` | Benchmark Rust Crates (small apps) | `-p turbopack-cli` bench target `small_apps` |
  | `benchmark-analyzer` | Benchmark Rust Crates (analyzer) | `-p turbopack-ecmascript` bench target `references` |
  | `benchmark-large` | Benchmark Rust Crates (large) *(only on `benchmark` label or `workflow_dispatch`)* | `-p turbopack -p turbopack-bench` (all targets) |

  Build command pattern: `cargo codspeed build -p <package> [<target>]`; run command: `cargo codspeed run -p <package> [<target>]`.
