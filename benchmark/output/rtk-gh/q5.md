## Answer

- `redirect()` is defined in `packages/next/src/client/components/redirect.ts` at line 38 (`export function redirect(`).
- It interrupts rendering by throwing: `throw getRedirectError(url, type, RedirectStatusCode.TemporaryRedirect)` (line 45). `getRedirectError` sets `error.digest = \`${REDIRECT_ERROR_CODE};${type};${url};${statusCode};\`` encoding the redirect destination.
- Caught in `packages/next/src/server/app-render/app-render.tsx` inside `async function renderToStream` (line 3147): `} else if (isRedirectError(err)) {` (line 3976) sets `res.statusCode = getRedirectStatusCodeFromError(err)` and `setHeader('location', redirectUrl)` to build the HTTP redirect response.
