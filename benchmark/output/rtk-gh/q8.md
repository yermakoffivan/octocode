## Answer

- Header: `ACTION_HEADER = 'next-action' as const` at `packages/next/src/client/components/app-router-headers.ts` line 2. The handler in `action-handler.ts` reads `req.headers.get(ACTION_HEADER)` to identify Server Action requests.
- Executor: `export async function handleAction` at `packages/next/src/server/app-render/action-handler.ts` line 538.
- Response building: when the action result is RSC content, it returns `return new FlightRenderResult(response.body!)` (line 281 in `action-handler.ts`). For non-streaming actions it returns `RenderResult.fromStatic('{}', JSON_CONTENT_TYPE_HEADER)` (line 300).
