Title is "Why Research Helpers". Four real helpers in the codebase: research_helper (octocode, cross-repo / multi-hop code), trino_helper (trino, multi-table SQL / lineage), grafana_helper (grafana, logs → request trace → deploy correlation), adaptive_helper (per-call MCP combination). Pros: reduces junk from parent context, leaves more reasoning room for the researcher. Cons: more code to maintain, more complex flow, extra round-trip adds latency to the final answer.
