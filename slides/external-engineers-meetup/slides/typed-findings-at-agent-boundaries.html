Typed findings beat prose at agent boundaries.

{
  "agent": "grafana_helper",
  "question": "Did the error rate spike correlate with the deploy?",
  "confidence": "verified",      // verified | likely | unverified
  "evidence": [
    {
      "ref": "dashboardUid/panelId + deploy annotation",
      "claim": "Error rate increased shortly after deployment."
    }
  ],
  "gaps": []                       // required — cannot be omitted
}

The parent receives decision-grade evidence, not text to interpret under uncertainty.

Confidence is explicit. Gaps are required. Helpers don't return loose prose and hope.
