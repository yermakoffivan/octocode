Rule 01 / Intent Goal, subgoal, why

The agent commits to what it's investigating before the call goes out — the result stays narrow instead of becoming a repo dump.

Rule 04 / Schema Typed inputs, no ambiguity

Strict zod-style schemas catch malformed calls before the network — the LLM can't accidentally ship junk.

Rule 05 / Parallel Fan out per call

One invocation, many queries — explore branches in parallel without paying for extra round-trips.
