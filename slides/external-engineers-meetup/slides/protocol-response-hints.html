Rule 02 / Hints Next call, or recovery

Hints translate raw results into a recommended next step — the agent doesn't have to invent the loop on its own.

Rule 03 / Descriptions When to use, what to avoid

Tool descriptions teach the LLM the right entry point and the gotchas before it ever picks a tool.

Rule 06 / Cache · Rule 07 / Pagination Cheap re-reads, safe big payloads

Large files and PRs come back paginated and cache locally — the loop stays fast and the context window stays whole.

Hints turn a raw result into a next step. Read both shapes: success-with-results and the empty-with-fallback case. Close on cache + pagination as the platform-side guarantees that keep the loop cheap.
